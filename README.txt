php artisan make:migration add_column_название колонки(слитно)_to_название таблицы_table
В схеме пишешь стандартно $table->значение колонки (название колонки)->nullable(); нулевое значение делается для того, чтобы не было конфликтов. Скобки можно не ставить, но поставь
$table->dropColumn(название колонки); это ролбечнит колонку
